GeometricAlgebra.Blade — Type
Blade{B,T}

GeometricAlgebra.grade_index — Method
`grade_index(dim, i)`

Return the grade index of i.

Example

julia> grade_index(3, [1])
1

julia> grade_index(3, [1, 2])
1

julia> grade_index(3, [3, 1])
3

GeometricAlgebra.@basis — Macro
@basis [mod=Main, prefix=:v, signature="Ø"] <dim>

Pull in all unit blade symbols from a dim-dimensional geometric algebra with a given signature. The symbols are evaluated inside the module mod, prefixed with prefix.
